IMPORTING NEW DATA into the Tanner Report Database

1. download the data; store in TannerXXXX, where XXXX is the data year. 

2. a. Open enclosed facilities list in gnumeric; replace ',' with '_'
   b. save as csv
   c. manage enclosed Methods listing as necessary.

3. Copy read_fac_xxxx.m into the appropriate directory and modify it to be
correct. 

4. a. Run read_fac_xxxx
   b. save new Facilities file

5. Create a new entry in mdmfa.m for the new year, if necessary

6. run mdmfa(<Tanner-file>,<year>,<wastecode>)
